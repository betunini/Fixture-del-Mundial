# Fixture-del-Mundial
Python application that generates a round-robin football fixture, records match results, and updates league standings automatically.
# Football Fixture Manager

A console-based application developed in Python that simulates a football league.

## Features

- Register four teams
- Generate a complete round-robin fixture
- Record match results
- Update league standings automatically
- Display statistics for each team

## Skills demonstrated

- Python fundamentals
- Functions
- Lists and nested lists
- Loops (for / while)
- Conditional statements
- Input validation
- Menu-driven programs

## Future improvements

- Save data to files
- Support any number of teams
- Add a graphical interface
